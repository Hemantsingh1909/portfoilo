# Hemant Satwal — Portfolio Website

My personal portfolio, built to showcase my work as a Software Engineer specializing in full-stack development and AI/ML.

**Live site:** [hemantsatwal.in](https://hemantsatwal.in)

## About

This site highlights my background, skills, and projects — including work from my time at Publicis Sapient, my SDE internship at Bluestock Fintech, and self-initiated projects like [ATSPrime](https://atsprime.in), an AI-powered resume tailoring tool.

## Tech Stack

- **Framework:** React + Vite
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui, Radix UI
- **Animations:** Framer Motion
- **Contact Form:** EmailJS
- **Deployment:** Vercel

## Features

- Responsive, dark-mode-first design
- Smooth scroll navigation across Home, Skills, Experience, Projects, and Contact sections
- Interactive project showcase with live deployed links
- Downloadable resume
- Working contact form with real-time email delivery

## Running Locally

Clone the repository and install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The site will be available at `http://localhost:5173`.

### Environment Variables

To enable the contact form, create a `.env` file in the project root with your EmailJS credentials:

```
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

## Building for Production

```bash
npm run build
```
